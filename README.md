Para lanzar el proyecto hay que haber cumpkido con varios requisitos anteriores:
1. Descargar el proyecto
2. Descomprimir la carpeta
3. Descargar Docker Desktop

Tras ello, nos vamos a la ruta en la que se encuentre el proyecto, exactamente a la misma altura que el archivo Dockerfile; 
y debemos de ejecutar el comando docker-compose up --build.
